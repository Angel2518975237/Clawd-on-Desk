# Code of Conduct

## Our Standard

We want this project to be practical, respectful, and easy to collaborate on.

Examples of behavior that help:

- giving constructive technical feedback
- respecting different experience levels
- discussing tradeoffs without personal attacks
- reporting bugs clearly and honestly

Examples of unacceptable behavior:

- harassment or intimidation
- insulting or mocking contributors
- publishing private information without permission
- bad-faith disruption of discussions or reviews

## Enforcement

Project maintainers may remove comments, close threads, or block participation when behavior harms the project or other contributors.

## Scope

This applies to issues, pull requests, discussions, and other project communication channels.
